◆ checkId()

_Bool checkId ( int  id)

Checks if the given ID is legal according to the Luhn algorithm.

Parameters
id

Overview of algorithm :

  • From the rightmost digit, and moving left, double the value of every second digit
  • Sum all the digits of the modified number
  • If the total mod 10 is 0, the id is valid.
